# 📝 DevBlog

Un blog simple desarrollado como parte del bootcamp de Generation.
Permite crear y visualizar publicaciones de forma dinámica en el navegador.


---

## 📌 Descripción

DevBlog es una aplicación web básica construida con tecnologías front-end que simula la creación de publicaciones tipo blog.

El objetivo principal del proyecto es practicar:

* Manipulación del DOM
* Manejo de eventos en JavaScript
* Uso de componentes de Bootstrap
* Estructuración básica de una interfaz web

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

---

## ⚙️ Funcionalidades

* Crear nuevas publicaciones mediante un formulario
* Visualizar publicaciones en pantalla
* Interfaz simple y responsiva
* Navegación básica (Inicio, About, Contact)

---

---

## 📚 Aprendizajes

Durante este proyecto se reforzaron conceptos clave como:

* Manipulación del DOM con JavaScript
* Captura y manejo de eventos
* Organización básica de archivos
* Uso de Bootstrap para maquetación rápida
* Separación de estructura, estilo y lógica

---

## ⚠️ Limitaciones actuales

* No cuenta con persistencia de datos (los posts se pierden al recargar)
* No hay backend ni base de datos
* Funcionalidad limitada a entorno local

---

## 🔮 Posibles mejoras

* Implementar almacenamiento con `localStorage`
* Añadir backend (Node.js / Firebase)
* Sistema de autenticación
* Edición y eliminación de posts
* Soporte para Markdown
* Mejoras en UI/UX (tema personalizado)
